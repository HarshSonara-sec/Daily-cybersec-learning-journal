# Linux Troubleshooting

## Issue

Recurring system lag/slowness while using Kali Linux.

Baseline hardware checks indicated that storage and memory were healthy.

## Findings

- Disk Usage: 57%
- RAM Usage: 1.7 GB / 7.5 GB
- Swap Usage: 0 GB
- fstrim.timer: Active and Enabled
- 11 GNOME Shell Extensions were running simultaneously

Likely cause:

Multiple GNOME Shell extensions were increasing compositor overhead, resulting in desktop lag.

---

## Commands Used

```bash
df -h

free -h

swapon --show

systemctl status fstrim.timer

gnome-extensions list --enabled

gnome-extensions disable system-monitor@gnome-shell-extensions.gcampax.github.com

gnome-extensions disable tiling-assistant@leleat-on-github
```

---

## Resolution

Disabled the following GNOME Shell extensions:

- System Monitor
- Tiling Assistant

Result:

Reduced GNOME compositor load which should improve desktop responsiveness.
