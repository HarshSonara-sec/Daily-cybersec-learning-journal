## Key Takeaways

* Block Size = 256 − Last Subnet Mask Octet.
* Broadcast Address = Next Network − 1.
* First Host = Network + 1.
* Last Host = Broadcast − 1.
* In VLSM, always allocate the largest subnet first.
* Never overlap subnet ranges.
