# What I Learned Today

- Ethernet uses Frames instead of Packets.
- Switches forward traffic using MAC addresses rather than IP addresses.
- ARP translates IPv4 addresses into MAC addresses within a local network.
- ICMP Ping verifies connectivity using Echo Request and Echo Reply messages.
- Unknown unicast traffic is flooded until the switch learns the destination MAC address.
