## Linux Architecture

Learned the difference between:

- x64
- AMD64
- ARM

Key Notes:

- AMD64 and x86_64 refer to the same 64-bit architecture.
- ARM uses a different CPU architecture optimized for efficiency and low power consumption.
- Most cybersecurity tools support x86_64 first.
