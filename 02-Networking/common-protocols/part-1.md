# Common Network Protocols - Part 1

| Sr. No. | Protocol                              | Acronym | Default Port(s) | Description                                           |
| :-----: | ------------------------------------- | :-----: | :-------------: | ----------------------------------------------------- |
|    1    | HyperText Transfer Protocol           |   HTTP  |      80/TCP     | Transfers web pages between client and server.        |
|    2    | HyperText Transfer Protocol Secure    |  HTTPS  |     443/TCP     | Encrypted version of HTTP using TLS/SSL.              |
|    3    | File Transfer Protocol                |   FTP   |    20,21/TCP    | Transfers files between systems.                      |
|    4    | Trivial File Transfer Protocol        |   TFTP  |      69/UDP     | Lightweight file transfer without authentication.     |
|    5    | SSH File Transfer Protocol            |   SFTP  |      22/TCP     | Secure file transfer over SSH.                        |
|    6    | Secure Shell                          |   SSH   |      22/TCP     | Secure remote administration.                         |
|    7    | Telnet                                |  Telnet |      23/TCP     | Insecure remote terminal access.                      |
|    8    | Simple Mail Transfer Protocol         |   SMTP  |      25/TCP     | Sends email between mail servers.                     |
|    9    | Post Office Protocol Version 3        |   POP3  |     110/TCP     | Downloads email to the client.                        |
|    10   | Internet Message Access Protocol      |   IMAP  |     143/TCP     | Synchronizes email across devices.                    |
|    11   | Domain Name System                    |   DNS   |    53/TCP,UDP   | Resolves domain names to IP addresses.                |
|    12   | Dynamic Host Configuration Protocol   |   DHCP  |    67,68/UDP    | Automatically assigns IP addresses.                   |
|    13   | Server Message Block                  |   SMB   |     445/TCP     | Windows file and printer sharing.                     |
|    14   | Network File System                   |   NFS   |   2049/TCP,UDP  | Linux/Unix file sharing.                              |
|    15   | Lightweight Directory Access Protocol |   LDAP  |   389/TCP,UDP   | Accesses directory services such as Active Directory. |
